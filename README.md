# [Oren Lyons Archive](http://orenlyonsarchive.com/)


### How to
+ Setup configuration on `_data/metadata.yaml`
+ Hero setup on `_data/hero.yaml` you can disable hero by switch show from true to be false
+ Section area setup on `_data/section.yaml`
+ Gallery setup on `_data/gallery.yaml`
+ Fouding setup on `_data/founding.yaml`
+ Sponsor setup on `_data/sponsor.yaml`
+ Navbar area setup on `_data/navbar.yaml`
+ Footer setup on `_data/footer.yaml`
